# @appandflow/rn-magic-scroll

Library to help managing keyboard interaction in form

## Installation

```sh
npm install @appandflow/rn-magic-scroll
```

## Usage

```js
import { multiply } from '@appandflow/rn-magic-scroll';

// ...

const result = await multiply(3, 7);
```
